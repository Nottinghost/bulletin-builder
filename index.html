Which button to use: Print — Full Layout makes the traditional 3-column sheet, landscape — best for printing at church or reading on a computer. Save PDF — Phone Layout reflows everything into one column, portrait — best for texting or emailing to someone who'll open it on their phone.

To print or save: click a button above, then choose Letter paper in the print window (the Full Layout switches orientation to landscape automatically). If it runs onto a second page, that's normal for a bulletin — print double-sided and it's one folded sheet.

To email or text it: in the print window, set the destination to Save as PDF and save the file, then attach it. The PDF keeps the design perfectly for everyone who opens it.

Important: if the navy and gold print white, turn on “Background graphics” (or “Print backgrounds”) in the print window — it's usually near “More settings.”

The church logo is already built in. You only need the Logo box under “Church info” if you ever want to swap it.
